
## 使用方法

1. アプリケーションを起動します。
2. コンボボックスから使用するMIDIデバイスを選択します。
3. データグリッドビューにMIDIノートと対応するキーを入力します。
4. 「Save」ボタンをクリックしてマッピングを保存します。
5. 「Load」ボタンをクリックしてマッピングを読み込みます。

## 初期設定

アプリケーション初回起動時に `mappings.txt` ファイルが存在しない場合、初期値が書き込まれたファイルが自動的に生成されます。初期値の例として、以下のマッピングが含まれます：

- MIDIノート60 -> キーA
- MIDIノート62 -> キーB

## 開発

### コミットメッセージ

初期化時に LoadMappings を呼び出し、mappings.txt が存在しない場合に初期値を書き込むロジックを追加

- コンストラクタで LoadMappings メソッドを呼び出すように変更
- LoadMappings メソッドでファイルが存在しない場合に初期値を書き込むロジックを追加
- 初期値の例として、MIDIノート60をキーAに、MIDIノート62をキーBにマッピング
